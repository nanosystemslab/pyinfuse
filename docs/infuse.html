Introduction

PyInfuse provides a Python interface for controlling syringe pumps via a serial port connection. It simplifies the process of managing pumps daisy-chained together by offering a high-level API for tasks such as setting flow rates, syringe diameters, and controlling the infusion or withdrawal of fluids.

Installation

To install PyInfuse, you can use pip:

pip install pyinfuse

Quick Start

This section walks you through the basic steps of setting up a pump chain, configuring a pump, and starting an infusion. 

from pyinfuse import Chain, Pump

# Initialize the chain on a specific serial port
chain = Chain('/dev/ttyUSB0')

# Add a Pump to the chain
pump = Pump(chain, address=1, name='Pump 11')

# Set syringe diameter and flow rate
pump.setdiameter('25')
pump.setflowrate('5', 'uL/min')

# Start infusing
pump.infuse()

API Reference

Chain

class pyinfuse.Chain(port: str)

Create Chain object.

Harvard syringe pumps are daisy chained together in a ‘pump chain’ off a single serial port. A pump address is set on each pump. You must first create a chain to which you then add Pump objects.

Chain is a subclass of serial.Serial. Chain creates a serial.Serial instance with the required parameters, flushes input and output buffers (found during testing that this fixes a lot of problems) and logs creation of the Chain.

Pump

class pyinfuse.Pump(chain: Chain, address: int = 0, name: str = 'Pump 11')

Create Pump object for Harvard Pump 11.

Argument:

Chain: pump chain

Optional arguments:

address: pump address. Default is 0. name: used in logging. Default is Pump 11.

infuse() → None

Start infusing pump.

setdiameter(diameter: str) → None

Set syringe diameter (millimetres).

Pump 11 syringe diameter range is 0.1-35 mm. Note that the pump ignores precision greater than 2 decimal places. If more d.p. are specificed the diameter will be truncated.

setflowrate(flowrate: str, units: str) → None

Set flow rate (microlitres per minute).

Flow rate is converted to a string. Pump 11 requires it to have a maximum field width of 5, e.g. “XXXX.” or “X.XXX”. Greater precision will be truncated.

The pump will tell you if the specified flow rate is out of range. This depends on the syringe diameter. See Pump 11 manual.

settargettime(targettime: int) → None

Set the target time to infuse or withdraw (sec).

settargetvolume(targetvolume: str, units: str) → None

Set the target volume to infuse or withdraw (microlitres).

stop() → None

Stop pump.

withdraw() → None

Start withdrawing pump.

Exceptions

exception pyinfuse.PumpError

Examples

The following example demonstrates how to control multiple pumps in a chain. 

chain = Chain('/dev/ttyUSB0')
pump1 = Pump(chain, address=1, name='Pump 1')
pump2 = Pump(chain, address=2, name='Pump 2')

# Configure the pumps
pump1.setdiameter('12.5')
pump1.setflowrate('100', 'uL/min')
pump2.setdiameter('25')
pump2.setflowrate('200', 'uL/min')

# Start both pumps
pump1.infuse()
pump2.infuse()

Conclusion

PyInfuse offers an efficient way to control syringe pumps through serial communication. By abstracting the low-level details, it enables researchers and developers to focus more on their experiments and less on the intricacies of hardware communication.
